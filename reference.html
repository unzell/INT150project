A SUMMARY OF CSS RULES THAT ARE NOT COVERED IN CLASS

1. VIDEO TAG

SOURCE: MDN video

RULES: embed a video attributes like autoplay, loop, muted, and playsinline to control how the video plays

MY IMPLEMENTATION: Used as a full-screen background video

2. EXTERNAL LINK

SOURCE: MDN link

RULES: links external resources like stylesheets to the document

MY IMPLEMENTATION: Imported Google Fonts (Noto Serif JP) and Font Awesome icons

3. :ROOT CSS PSEUDOCLASS

SOURCE: MDN :root

RULES: represents as the top-level element that can be used to store global CSS variables

MY IMPLEMENTATION: unified variables for color modification across the site (i used it to declare the main theme of the site --main --white etc.)

        :root {
            --primary-accent: #ff7f50;
        }

4. CLIP PATH

SOURCE: MDN clip-path

RULES: creates a clipping shape to hide parts of an element like the shape of octogon

MY IMPLEMENTATION: used to cut the corners of the team member name plate as stylized hexagon shape as seen in the team member section

        clip-path: polygon(
            0% 2rem,
            2rem 0%,
            100% 0,
            100% calc(100% - 2rem) ,
            calc(100% - 2rem) 100%,
            0% 100%
        );

5. BACKDROP FILTER

SOURCE: MDN backdrop-filter

RULES: applies graphical effects (like blur) to the area behind an element

MY IMPLEMENTATION: used to created a glass-like blur effect on the team member name plate over the background video

        backdrop-filter: blur(12px);

6. HOVER CURSOR

SOURCE: Changes the mouse pointer when interacting with an element

MY IMPLEMENTATION: made footer navigation arrow shows a pointer hand when hovered over (originally it was just a div box with no pointer when hovered)

        cursor: pointer;
